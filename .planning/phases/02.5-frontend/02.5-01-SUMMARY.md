## Decisions Made

- **@vitejs/plugin-react v4 instead of v6:** The plan specified v6.0.1 but it requires Vite 8. Since the project is on Vite 5, pinned to @vitejs/plugin-react@^4.7.0 (compatible with Vite 4/5/6/7).
- **Manual components.json instead of interactive shadcn init:** The current shadcn CLI uses a preset-based workflow (Nova, Vega, etc.) with no Tailwind v3 compatibility prompts. Created components.json manually and used `shadcn add` directly, which successfully detected our Tailwind v3 setup.
- **Frontend .gitignore override:** Root .gitignore had `lib/` pattern that excluded frontend/src/lib/utils.ts. Added `!src/lib/` negation rule to frontend's .gitignore.
