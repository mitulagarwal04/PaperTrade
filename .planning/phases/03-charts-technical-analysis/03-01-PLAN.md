## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | `/api/v1/charts/{symbol}` path param `symbol` | mitigate | yfinance's `download()` handles invalid symbols gracefully (returns empty DataFrame). FastAPI path param is typed str. |
| T-03-02 | Tampering | `range` and `interval` Query params | mitigate | FastAPI Query param regex validation restricts to known values (`^(1d\|5d\|1mo\|3mo\|6mo\|1y\|5y)$` and `^(1m\|5m\|15m\|30m\|1h\|1d\|1wk)$`). Invalid values return 422 before reaching handler. |
| T-03-03 | Denial of Service | yfinance blocking call in executor | mitigate | `run_in_executor` prevents event loop blocking. CacheManager with TTL reduces redundant requests. |
| T-03-04 | Information Disclosure | yfinance response contains full OHLC data | accept | All market data is publicly available via Yahoo Finance. No PII is involved. Single-user application. |
</threat_model>
